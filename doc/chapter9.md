# DRF-React Study

## Chapter 9. 리액트 기초 익히기

### 개발환경 세팅

- node.js 설치

  - 12.16.1 버전 설치

  - 본인은 docker 이미지 사용

    ```bash
    docker run -it --rm -d node:12.16.1
    ```

  - yarn 설치

    ```bash
    npm install yarn
    ```

  - react 프로젝트 설치

    ```bash
    yarn create react-app [프로젝트명]
    ```
